# typescript-koa

## Use

>   git clone https://github.com/qqxs/typescript-koa.git

>   cd typescript-koa && yarn install

>   yarn run build

>   yarn run server

如果开发使用

>   yarn run watch-server

## views

渲染模版使用`pug`



## 错误信息格式

```json
{
    "code": 404,
    "message": "this is error msg"
}
```
